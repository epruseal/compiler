# 대한민국 행정규칙 저장소

대한민국 행정규칙을 Git 저장소로 관리합니다. 각 행정규칙은 Markdown 파일이고, 원문 XML에서 추출한 메타데이터와 본문을 함께 기록합니다.

[legalize-kr](https://github.com/legalize-kr/legalize-kr) (대한민국 법령 Git 저장소)의 자매 프로젝트입니다.

> **공지**: 행정규칙 수집/변환 파이프라인이 개선될 경우, 전체 저장소를 재구성하기 위해 force-push가 실행될 수 있습니다. 이 경우 모든 commit hash가 변경됩니다. 이 저장소를 fork하거나 참조하는 경우, force-push 이후 `git fetch --all && git reset --hard origin/main`으로 동기화해 주세요.

## 빠른 시작

```bash
git clone https://github.com/legalize-kr/admrule-kr.git
cd admrule-kr

# 특정 부처의 행정규칙 보기
ls 행정안전부/

# 고시 목록 보기
ls 행정안전부/_본부/고시/

# 전체 행정규칙에서 특정 단어 검색
grep -rl "개인정보" .
```

## 구조

```text
{기관경로...}/
  {행정규칙종류}/
    {행정규칙명}/
      본문.md
    {행정규칙명}_{발령번호 또는 행정규칙일련번호}/
      본문.md
```

예:

```text
행정안전부/_본부/고시/공공데이터 관리지침/본문.md
국토교통부/제주지방항공청/훈령/제주지방항공청 사무분장 및 위임전결 사항 등에 관한 규정/본문.md
보건복지부/국립소록도병원/훈령/국립소록도병원 당직 및 비상근무 규정/본문.md
농림축산식품부/산림청/국립산림과학원/훈령/국립산림과학원 위임전결 규정/본문.md
국무총리/법제처/훈령/법제처 사무분장규정/본문.md
대통령/국가교육위원회/고시/국가교육위원회 운영에 관한 규칙/본문.md
대통령/방송미디어통신위원회/방송미디어통신사무소/훈령/방송미디어통신사무소 사무분장 등에 관한 세칙/본문.md
과학기술정보통신부/국립전파연구원/고시/전자파적합성 기준/본문.md
과학기술정보통신부/중앙전파관리소/훈령/중앙전파관리소 사무분장세칙/본문.md
세월호 선체조사위원회/_본부/훈령/세월호 선체조사위원회 정보공개 운영규정/본문.md
```

경로 앞부분은 법령상 확인되는 기관 계층입니다. 실제 발령 주체가 최상위 본부와 같으면 `_본부`를 사용하고, 외청·위원회·처·공공기관·소속기관처럼 별도 발령 주체가 있으면 확인되는 상위기관부터 발령 주체까지를 차례로 둡니다. 동일한 기관경로·종류·이름이 충돌하는 경우 발령번호 또는 행정규칙일련번호를 파일 경로에 붙여 충돌을 해소합니다.

## Markdown 형식

각 파일은 YAML frontmatter가 포함된 Markdown입니다.

```yaml
---
행정규칙ID: '...'
행정규칙일련번호: '...'
행정규칙명: '공공데이터 관리지침'
행정규칙종류: '고시'
상위기관명: '행정안전부'
소관부처명: '행정안전부'
기관경로:
- '행정안전부'
발령번호: '...'
발령일자: 2024-05-04
시행일자: 2024-05-05
제개정구분: '일부개정'
본문출처: 'api-text'
출처: 'https://www.law.go.kr/행정규칙/공공데이터관리지침'
---

본문 내용
```

`본문출처: parsing-failed`인 문서는 국가법령정보센터 XML에 본문 조문이 없고, 원문이 첨부파일로 제공되는 경우입니다.

## 기관명과 경로 정규화

행정규칙의 기관 경로는 국가법령정보센터 OpenAPI의 `상위부처명`, `소관부처명`, `담당부서기관명`을 정규화한 뒤, 정부조직법 및 개별 설치 법률에서 확인되는 상위기관 관계를 반영합니다. 원천 데이터에는 기관명 표기 오류, 가운뎃점 표기 차이, 정부조직 개편에 따른 과거 기관명, `정부산하기관및위원회` 같은 포괄 분류가 함께 존재하므로 다음 원칙을 적용합니다.

### 기본 정리

- Unicode NFC 정규화를 적용합니다.
- 경로 구분자(`/`, `\`, `:` 등)는 공백으로 바꾸고 연속 공백은 하나로 줄입니다.
- `소관부처명`이 날짜처럼 들어온 명백한 원천 오류(`YYYY-MM-DD`)는 `상위부처명`으로 대체합니다.
- `10.29이태원참사진상규명과재발방지를위한특별조사위원회`는 `10·29이태원참사진상규명과재발방지를위한특별조사위원회`로 통일합니다.

### 경로 결정 원칙

- 지방청·소속기관·소속 병원·사무소·담당관 등은 확인되는 기관 계층 아래에 둡니다. 예를 들어 `원주지방환경청`은 `기후에너지환경부/원주지방환경청/...`, `목포해양경찰서`는 `해양수산부/해양경찰청/목포해양경찰서/...`가 됩니다.
- `상위부처명`이 `기후에너지환경부 국립환경과학원`처럼 `상위기관 + 소속기관` 형태로 들어오고, `담당부서기관명`이 `국립환경과학원(환경보건정책과)`처럼 그 소속기관을 확인해 주는 경우에는 `기후에너지환경부/국립환경과학원/...`으로 분리합니다.
- 기능이 나뉜 과거 부처명은 단일 치환하지 않되, 과거 분할 부처 또는 공식적으로 확인된 기능 이관 쌍에서 `담당부서기관명`이 현재의 본부급 담당기관을 명확히 가리키는 경우에만 그 기관으로 보정합니다. 예를 들어 `국토해양부` 문서라도 `담당부서기관명`이 `해양수산부(해양영토과)`이면 `해양수산부/_본부/...` 아래에 두고, 2025년 개편으로 산업통상부의 에너지 기능이 기후에너지환경부로 이관된 범위는 `기후에너지환경부/_본부/...`로 둡니다. 반대로 현재 본부급 기관의 문서에 다른 기관이 협조·담당부서로 등장한다는 이유만으로 발령기관을 바꾸지는 않습니다.
- `담당부서기관명`이 `기후에너지환경부(하천계획과)`처럼 하천 업무의 현재 본부급 담당기관을 명확히 가리키는 경우에는 원천의 `소관부처명`이 `국토교통부`여도 `기후에너지환경부/_본부/...` 아래에 둡니다.
- 원천 API의 `상위부처명`이 이미 현행 본부급 부처이고 `소관부처명`만 과거 본부급 부처명인 경우에는 그 과거명을 산하기관으로 보지 않고 현행 본부로 접습니다. 예를 들어 `농림축산식품부/농림수산식품부/...`는 `농림축산식품부/_본부/...`로, 수산 분야 문맥의 `해양수산부/농림수산식품부/...`는 `해양수산부/_본부/...`로 둡니다.
- 정부조직법이나 개별 설치 법률이 외청·위원회·처·소속기관의 상위기관을 명시하면 그 기관 계층을 경로에 반영합니다. 예를 들어 `병무청`은 `국방부/병무청/...`, `국세청`과 `관세청`은 `재정경제부/국세청/...`, `재정경제부/관세청/...`, `산림청`은 `농림축산식품부/산림청/...`, `법제처`는 `국무총리/법제처/...`가 됩니다.
- 개별 법률이 대통령 또는 국무총리 소속 위원회로 설치한 중앙행정기관도 그 상위기관을 반영합니다. 예를 들어 `국가교육위원회`와 `방송미디어통신위원회`는 `대통령/...`, `공정거래위원회`, `금융위원회`, `국민권익위원회`, `개인정보보호위원회`, `원자력안전위원회`는 `국무총리/...` 아래에 둡니다.
- `국립전파연구원`과 `중앙전파관리소`는 서로 다른 기관입니다. 현행 [과학기술정보통신부와 그 소속기관 직제](https://www.law.go.kr/법령/과학기술정보통신부와그소속기관직제) 제2조는 두 기관을 모두 과학기술정보통신부장관 소속기관으로 둡니다. 반면 [방송미디어통신위원회와 그 소속기관 직제](https://www.law.go.kr/법령/방송미디어통신위원회와그소속기관직제) 제2조는 방송미디어통신위원회 소속기관으로 `방송미디어통신사무소`를 둡니다.
- 독립성이 법률상 명시되어 있으나 특정 상위기관 소속으로 규정되지 않은 기관은 흡수하지 않습니다. 예를 들어 [국가인권위원회법](https://www.law.go.kr/법령/국가인권위원회법) 제3조의 `국가인권위원회`, [고위공직자범죄수사처 설치 및 운영에 관한 법률](https://www.law.go.kr/법령/고위공직자범죄수사처설치및운영에관한법률) 제3조의 `고위공직자범죄수사처`, `중앙선거관리위원회`, [10ㆍ29이태원참사 피해자 권리보장과 진상규명 및 재발방지를 위한 특별법](https://www.law.go.kr/법령/10ㆍ29이태원참사피해자권리보장과진상규명및재발방지를위한특별법) 제6조·제7조의 `10·29이태원참사진상규명과재발방지를위한특별조사위원회`, [세월호 선체조사위원회의 설치 및 운영에 관한 특별법](https://www.law.go.kr/법령/세월호선체조사위원회의설치및운영에관한특별법) 제3조·제4조·제9조의 `세월호 선체조사위원회`는 별도 루트로 유지합니다.
- `정부산하기관및위원회`처럼 포괄 분류로 들어온 문서는 개별 법률로 상위기관이 확인되는 경우에만 이동합니다. 예를 들어 `수도권매립지관리공사`는 [수도권매립지관리공사의 설립 및 운영 등에 관한 법률](https://www.law.go.kr/법령/수도권매립지관리공사의설립및운영등에관한법률) 제5조·제7조에 따라 `기후에너지환경부/수도권매립지관리공사/...`, `평생교육진흥원`은 `국가평생교육진흥원`으로 정규화한 뒤 [평생교육법](https://www.law.go.kr/법령/평생교육법) 제19조에 따라 `교육부/국가평생교육진흥원/...` 아래에 둡니다.

### 과거 조직명

정부조직 개편에 따른 과거 기관명은 무조건 현재 기관명으로 합치지 않습니다. 같은 `기관코드` 아래에도 본부, 소속기관, 위원회, 과거 부처명이 섞여 있고, 한 부처의 기능이 여러 기관으로 나뉘는 경우가 있기 때문입니다.

병합은 다음 조건을 모두 만족할 때만 적용합니다.

1. 정부조직법 개정, 부처 보도자료, 행정규칙 일괄개정 등 공식 자료로 승계 관계가 확인됩니다.
2. 단순 명칭 변경 또는 포괄 승계에 가깝고, 기능이 둘 이상의 기관으로 분할되지 않습니다.
3. 같은 행정규칙의 식별자·명칭·발령 이력이 이어져 경로 병합이 이력 추적을 개선합니다.
4. 병합 후에도 원천의 `소관부처명` 또는 과거 기관명 정보를 frontmatter에서 확인할 수 있습니다.

현재 코드에 적용된 과거 조직명 매핑은 다음과 같습니다. 매핑된 문서는 경로와 `소관부처명`에 새 기관명을 사용하고, 원천 기관명은 `소관부처명_원문`에 보존합니다.

### 폐지ㆍ분할 기관 처리

폐지된 기관은 단순히 현재 이름 하나로 치환하지 않습니다. `국민안전처`, `국토해양부`, `농림수산식품부`, `교육과학기술부`처럼 기능이 둘 이상의 현행 기관으로 나뉜 경우에는 원천의 `상위부처명`, `소관부처명`, `담당부서기관명`이 현재 담당 기관을 확인해 주는 범위에서만 보정합니다.

예를 들어 `국민안전처`는 2014.11.19 정부조직법 개정으로 국무총리 소속 중앙행정기관으로 신설되었으나, 2017.7.26 정부조직법 개정으로 폐지되면서 재난안전 기능은 행정안전부로, 소방 기능은 소방청으로, 해양경비ㆍ안전 기능은 해양수산부 소속 해양경찰청으로 나뉘었습니다. 따라서 `국민안전처` 전체를 `행정안전부`로 일괄 병합하지 않고, 현재 [행정안전부와 그 소속기관 직제](https://www.law.go.kr/법령/행정안전부와그소속기관직제) 제2조에서 행정안전부장관 소속으로 확인되는 `국가재난안전교육원`과 `국립재난안전연구원` 같은 소속기관만 `행정안전부/...` 경로로 둡니다.

### 자동 병합하지 않는 경우

- 기능이 여러 기관으로 나뉜 개편은 단일 치환하지 않습니다. 예를 들어 [정부조직법](https://www.law.go.kr/법령/정부조직법) 개정으로 해양수산부가 신설되면서 구 `국토해양부`의 해양·해운항만 분야와 구 `농림수산식품부`의 수산 분야가 나뉜 경우 `국토해양부 → 국토교통부` 같은 자동 병합은 하지 않습니다. 다만 원천의 `담당부서기관명`이 `해양수산부(해양영토과)`처럼 현재 담당 본부급 기관을 직접 확인해 주는 문서는 그 기관으로 보정합니다.
- 본부와 소속기관의 구분: `환경부`, `국립환경과학원`, `한강유역환경청`처럼 같은 상위 계열이라도 행정규칙 발령 주체가 다르면 별도 경로로 유지합니다.
- 한시조직·위원회: `10·29이태원참사진상규명과재발방지를위한특별조사위원회`처럼 독립 발령 주체인 경우 현재 부처로 흡수하지 않습니다. 10·29이태원참사 특별법은 조사위원회의 설치와 독립 수행을 규정하지만 행정안전부 소속으로 두지 않고, 세월호 선체조사위원회 특별법도 위원회의 설치·독립성·위원 직무상 독립을 규정하지만 해양수산부 소속으로 두지 않습니다.
- 독립기관: 국가인권위원회법 제3조, 고위공직자범죄수사처법 제3조처럼 독립 수행이 명시되고 특정 상위기관 소속 조항이 없는 기관은 대통령·국무총리·부처 산하로 자동 편입하지 않습니다.
- 과거 `방송통신위원회` 발령 규칙이라도 현재 담당부서가 `과학기술정보통신부(주파수정책과)`처럼 과학기술정보통신부 본부로 확인되는 전파·통신 규칙은 [방송미디어통신위원회의 설치 및 운영에 관한 법률](https://www.law.go.kr/법령/방송미디어통신위원회의설치및운영에관한법률) 제3조의 대통령 소속 위원회로 자동 이동하지 않습니다.
- 원천 데이터만으로 승계 관계가 불명확한 경우: 매핑하지 않고 원문 기관명을 보존합니다.
